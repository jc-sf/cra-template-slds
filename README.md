# cra-template-slds

The SLDS template for [Create React App](https://github.com/facebook/create-react-app)

## Node version
>= 12
## Usage

- Create and initialize the app `my-app`:

  ```sh
  nvm use v12
  npx create-react-app my-app --template slds
  ```

- Setup SLDS
  ```sh
  cd my-app
  yarn setup-slds
  ```
